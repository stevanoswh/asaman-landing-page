---
title: "2. How can I create an Activity Container?"
published: true
---

**a.** In the left-hand menu, go to the **Activities** tab.  
Click the **“+ Add Container”** button in the top-right corner to create a new **Activity Container**.

![Open Activities and click “+ Add Container”](/help/3-2-1.jpg)

**b.** A modal will appear where you can add **Activity Container** details and set its **Allocation Type**.

![Fill details and choose Allocation Type](/help/3-2-2.jpg)

**c.** You can also set **Permissions** for an Activity Container.

![Set permissions for the container](/help/3-2-3.jpg)
