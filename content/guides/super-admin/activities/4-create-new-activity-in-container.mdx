---
title: "4. How can I create a new Activity in a Container?"
published: true
---

**a.** In the left-hand menu, go to the **Activities** tab.  
Click the **“View Activities”** button for an Activity Container.

![Open Activities and click “View Activities”](/help/3-4-1.jpg)

**b.** Click the **“+ Add Activity”** button to add an Activity in that Activity Container.

![Click “+ Add Activity” inside the container](/help/3-4-2.jpg)

**c.** A modal will appear where you can add details of an Activity.  
You can also preview the details of this Activity in the right sidebar.

![Add Activity details and preview](/help/3-4-3.jpg)
