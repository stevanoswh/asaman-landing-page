---
title: "11. How can I add an External Provider user to my account?"
published: true
---

**a.** In the left-hand menu, go to the **User Management** tab, and click the **“+ Add User”** button in the top-right corner.  
A pop-up will appear where you can select the **External Provider** user type to add.

![Select External Provider role](/help/2-11-1.jpg)

---

**b.** A modal will appear where you can add details for the **External Provider** user.

![External Provider details modal](/help/2-11-2.jpg)
