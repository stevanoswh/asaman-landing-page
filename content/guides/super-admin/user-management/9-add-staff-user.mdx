---
title: "9. How can I add a Staff user to my account?"
published: true
sort: 90
updatedAt: "2025-01-15"
---

**a.** In the left-hand menu, go to the **User Management** tab, and click the **“+ Add User”** button in the top-right
corner. A pop-up will appear where you can select the **Staff** user type to add.

![Select Staff role](/help/2-9-1.jpg)

<br />

**b.** A modal will appear where you can add details for the **Staff** user.

![Add details for Staff](/help/2-9-2.jpg)
