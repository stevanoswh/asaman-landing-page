---
title: "2. How can I add a user to my account?"
published: true
---

Go to the **User Management** tab in the left-hand menu, and click the **“+ Add User”** button in the top-right corner.  
A pop-up will appear where you can select the **user type** to add (Student, Parent, Staff, or External Provider).
