---
title: "7. How can I add a Student user to my account?"
published: true
sort: 70
updatedAt: "2025-01-15"
---

**a.** In the left-hand menu, go to the **User Management** tab, and click the **“+ Add User”** button in the top-right
corner. A pop-up will appear where you can select the **Student** user type to add.

![Select the Student role](/help/2-7-1.jpg)

<br />

**b.** A modal will appear where you can add **Student** details.  
Or, you can click the **“Link Sibling”** button in the modal to select a **Student** user from the dropdown list as a **sibling**.  
Then, click the **“Link or Add Parents”** button to link an already added **Parent** user from the dropdown, or add a new **Parent** user to this **Student** user.

![Student details modal](/help/2-7-2.jpg)

<br />

**c.** Select the **Parent** user from the dropdown.

![Link or Add Parents modal](/help/2-7-3.jpg)

**d**. Or, add the Parent user details.

![Add Parent user details](/help/2-7-4.jpg)
