---
title: "3. How to check the Confirmed Activities for an Activity Container in Random Allocation?"
published: true
---

**a.** In the left-hand menu, go to the **Activities** tab. Click the **“View Activities”** button for an Activity Container with **Random Allocation** type.

![Activities → View Activities](/help/4-3-1.jpg)

**b.** Then go to the **Allocations Overview** step and click the **“Confirmed Activities”** tab.

![Allocations Overview → Confirmed Activities tab](/help/4-3-2.jpg)

**c.** Here you can see the list of **Confirmed Activities**.

![Confirmed Activities list](/help/4-3-3.jpg)

**d.** Open the dropdown for any activity to view and update the **Students**.

![Open activity dropdown → view/update students](/help/4-3-4.jpg)
